# Alexander Lee — Portfolio

Personal portfolio website built with React and Tailwind CSS.

**Live site:** https://alexawnder.github.io/alexander-portfolio

## About

Data Analyst and CS graduate from Cal Poly Pomona. This site showcases my projects, work experience, and technical skills in Python, SQL, Power BI, and data engineering.

## Tech Stack

- React 18
- Tailwind CSS
- GSAP (animations)
- GitHub Pages (deployment)

## Running locally

```bash
npm install
npm start
```

## Deploying

```bash
npm run deploy
```

Builds the app and pushes to the `gh-pages` branch automatically.
